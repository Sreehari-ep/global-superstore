# global-superstore
power bi dashboard
Overview
This dashboard provides a comprehensive view of global sales data from the Global Superstore dataset. It enables users to analyze shipping modes, regional performance, and market segmentation across countries.
Features
- Country Table (Left Panel): Lists countries with their corresponding region and market.
- Shipping Mode Pie Chart (Center Panel): Visualizes the distribution of order counts by shipping mode.
- Country Filter (Right Panel): Allows users to filter data by selecting specific countries.
Use Cases
- Identify dominant shipping methods globally.
- Explore regional and market-specific trends.
- Filter data to analyze performance by country.
